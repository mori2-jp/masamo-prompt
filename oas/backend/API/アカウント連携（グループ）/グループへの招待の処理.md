
# 処理
## 招待
Admin は、コーチを招待出来る。
招待用のリンクを発行して、そのリンクをコーチが踏むとグループに参加できる。

## メンバーの作成
Admin は、新しいユーザーアカウントを作成してメンバーとしてグループに招待出来る。

## メンバーのログインID
メンバーは、ログインID不要で利用出来る。
Admin がグループに対して　Member　を追加すると、ログインIDを持たないアカウントが作成される。
