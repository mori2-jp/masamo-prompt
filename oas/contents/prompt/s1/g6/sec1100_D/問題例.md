```

　→ 順序を考える（例：1位、2位、3位）  
　→ 組み合わせる順番に意味があるとき

　→ 順序を考えない（例：AとB、BとAは同じ）

　→ 式で表す  
　→ 組み合わせの数の表し方もある

4　数の使い方  
　→ 組み合わせの数の使い方を学ぼう

5　本数の表し方  
　→ 20mのひもを4本に分けると、1本の長さは何mになりますか。  
　→ このひもを10本等分すると、1本分は何mになりますか。
```

```
　並べるときは、表を使ってみましょう。

　コインの表と裏のように、1つ1つ順に並べて

★ 組み合わせ方①  
　選び方を整理して書くと、わかりやすくなります。

★ 組み合わせ方②  
　表を使って、同じ組み合わせが2回数えられない  
　ように注意して書きましょう。

★ 組み合わせ方③  
　同じ組み合わせにならないように、記号や図を  
　使って整理して考えるとよいです。
```
