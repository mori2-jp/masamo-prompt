# 問題意図
・わり算には「等分除」と「包含除」があることを理解し，簡単な問題を式にできる。

# 問題例
(包含除) 12個のあめを，1人に3個ずつあげると，何人に配れますか？
(等分除) 12個のあめを，3人で同じ数ずつ分けたいとき，1人あたり何個になりますか？
