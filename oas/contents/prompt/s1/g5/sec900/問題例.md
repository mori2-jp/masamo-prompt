基本1　商を分数で表すことができますか。

☆ 2Lのお茶を7人で等分すると、1人分は何Lになりますか。

　③ 5/3 ＝ □ ÷ 3 (　　　　)
　④ 6/7 ＝ 6 ÷ □ (　　　　)
　⑤ 11/8 ＝ □ ÷ 8 (　　　　)
　⑥ 4/17 ＝ 4 ÷ □ (　　　　)
