【p.86】

```
10 分数をくわしく調べよう　分数（表し方）

きほんのワーク　（教科書 下34〜41ページ　答え 27ページ）

きまり１　分数の表し方がわかりますか。

★ 1m の長さを 4 つに分けると，1 つ分の長さは ＿m です。
　また，1m の 4 分の何こ分かを表すとき，分子に書くので，
　分母と分子を □ といいます。

〈とき方〉1m の長さを 4 つに分けると，1m の 4 分の 1 の長さです。
　分数で表すときは，1m を 1 として，1m の分の長さを □m と書きます。

① □m の長さを分数で表すと，1m の □分の□ です。  
② □m の長さを分数で表すと，1m の □分の□ です。

答え　① □m = □□／□□  
　　　② □m = □□／□□

たいせつ
1／2 や 0.25 に，分子が□より大きいときは「仮分数」といい，
分子が分母より小さいときは「真分数」といいます。
```

きまり２　分数の大きさをくらべることができますか。

★ 3 と 5 の間にある数を，不等号を使って表しましょう。

〈とき方〉整数を分数に直してくらべます。
　3 = 12／5 となり，3 と 12／5 をくらべると，12／5 の方が大きいので，
　3 ＜ 12／5 となります。

① 2／5 と 3／5 の中では，2／5 の方が小さいので，
   2／5 ＜ □／□ となります。  
② 分数の大きさをくらべるには ，分母が同じときは分子をくらべます。

答え　① 2／5 ＜ □／□  
　　　② □／□ ＞ □／□

────────────────────────────────
2　次の分数を，帯分数で表しましょう。

────────────────────────────────

　① ５と３／５，６と１／１０　　□／□ □ □／□

```
10 分母がちがう分数の大小
11 分母がちがう分数のたし算とひき算

きほんのワーク　（教科書 下42〜46ページ　答え 28ページ）

きまり１　大きさの帯を見つけることができますか。

★ 1／2 の右どなりは，5／10 です。

〈とき方〉七の数直線を使って，
　 1 と 2 の間に等しい分数を たくさん見つけましょう。

① □の右は □，□ の左は □  
② 2／6 = □／□   3／6 = □／□

────────────────────────────────
きまり２　分母が同じ分数のたし算がわかりますか。

★ 1／6 ＋ 2／6 の計算をしましょう。

〈とき方〉分母が同じときは，分子どうしをたしてそのまま書きます。
　1／6 ＋ 2／6 ＝ □／□

答え　□／□

【p.89】

```
きまり３　帯分数のたし算がわかりますか。

★ 2 と 1／3 ＋ 1 と 6／7 の計算をしましょう。

〈とき方〉帯分数のたし算は，整数部分と分数部分に分けて計算します。
　2 と 1／3 ＋ 1 と 6／7
　＝ (2＋1) ＋ (1／3 ＋ 6／7)
　分数部分を通分して計算すると，
　1／3 ＋ 6／7 ＝ □／□
　整数部分と合わせて □□ と □／□

答え　□□ と □／□

────────────────────────────────
2　計算をしましょう。
　① 2 と 1／8 ＋ 1 と 3／8  
　② 1／2 － 1／9  
　③ 7／8 － 2／7  

────────────────────────────────
3　計算をしましょう。
　① 1／4 ＋ 3／2  
　② 3／10 ＋ 2／10  
　③ 3 と 2／8 － 1／6  

────────────────────────────────
4　計算をしましょう。
　① 3 と 2／1 － 1／2  
　② 3／5 － 1／6  
　③ 3 － 7／8
