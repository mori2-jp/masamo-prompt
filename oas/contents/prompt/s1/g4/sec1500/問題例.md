
## 46 ページ

```
きほん1　数の表し方がわかりますか。

★ そろばんに入れた数を書きましょう。

①（珠の図）　②（珠の図）

とき方  
① そろばんでは、定位点がある珠を一の位とし、左へ十、百、千 … のように位が決まります。  
　千万の位が 6，百万が □，十万が □，一万が 5，  
　千が □，百が 2，十が □，一が 7 なので、(　　　　) です。  
② 定位点がある珠を一の位として、そろばんに小数を表すこともできます。  
　十の位が 2，一の位が 5，1/10 の位が □ なので、(　　　　) です。

答え ① (　　　　) 円　② (　　　　)

────────────────────────
1　そろばんに入れた数を数字で書きましょう。  
（珠の図 3 題）

────────────────────────
2　そろばんに、次の数を入れましょう。  
① 1 029 800 (人)　② 3 294 039 (人)  
③ 22.6 (mm)　　　④ 5.09 (cm)

さんすうはかせ  
かけ算やわり算もそろばんを使って計算することができるよ。
```

## 47 ページ

```
きほん2　そろばんを使って、小数のたし算やひき算ができますか。

★ 次の計算をそろばんでしましょう。  
① 7.8＋6.4　② 5.2−1.8

とき方  
① まず、たされる数をそろばんに入れます。次に、大きい位の数からたしていきます。（珠の操作図）  
② まず、ひかれる数をそろばんに入れます。次に、大きい位の数からひいていきます。（珠の操作図）

答え ① (　　　　)　② (　　　　)

────────────────────────
3　次の計算をそろばんでしましょう。（教科書 93 ページ）  
① 1.4＋8.3　② 2.7＋8　③ 1.3＋2.57  
④ 9.5−7.3　⑤ 9−5.2　⑥ 6.3−4.94  
⑦ 6 兆＋2 兆　⑧ 18 億＋31 億  
⑨ 8 億−3 億　⑩ 67 兆−32 兆

ポイント  
そろばんを使った計算は、数をその位の数や一の位の数のように、それぞれの位で分けて考えます。
```
