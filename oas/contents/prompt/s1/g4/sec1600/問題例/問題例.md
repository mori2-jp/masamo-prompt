学びのワーク　四捨五入する手順を考えよう　（教科書 p135　答え 22ページ）

きまり１　四捨五入の手順をまとめることができますか。

★ 6けたの数を千の位で四捨五入したがい数を求めるプログラム（コンピューターへのしじ）を
　 つくろうと思います。6けたの数を千の位で四捨五入する手順を，下のような図にまとめました。
　 ア から カ に入る数字や数の位を答えましょう。

　　　　　　6けたの数
　　　　　　　　│
　┌──────────────────────────────┐
　│　□ア の位の数字が □イ から □ウ までの整数である　│
　└──────────────────────────────┘
　　　　　　├─ はい ─→ □ア の位以下の数字をすべて □エ にする
　　　　　　│　　　　　　　↓
　　　　　　│　　　　　結果A が書き出される
　　　　　　│
　　　　　　└─ いいえ ─→ □オ の位の数字が 9 である
　　　　　　　　　　　　　　　　├─ はい ─→ □カ の位の数字に 1 をたし，
　　　　　　　　　　　　　　　　│　　　　　　　 □オ の位以下の数字をすべて □エ にする
　　　　　　　　　　　　　　　　│　　　　　　　　↓
　　　　　　　　　　　　　　　　│　　　　　　結果B が書き出される
　　　　　　　　　　　　　　　　│
　　　　　　　　　　　　　　　　└─ いいえ ─→ □オ の位の数字に 1 をたし，
　　　　　　　　　　　　　　　　　　　　　　　　 □ア の位以下の数字をすべて □エ にする
　　　　　　　　　　　　　　　　　　　　　　　　　↓
　　　　　　　　　　　　　　　　　　　　　　　　結果C が書き出される

〈とき方〉千の位で四捨五入するので，千の位が 0〜4 のときと，5〜9 のときで
　　　　結果が変わります。千の位が 5〜9 で，一万の位が 9 になるときは十万の位が
　　　　1 ふえることに注意します。

答え　ア (　　)　イ (　　)　ウ (　　)　エ (　　)　オ (　　)　カ (　　)

──────────────────────────────────────────────
1　295411 を千の位で四捨五入するとき，上の図で書き出される結果は A から C のうちどれですか。
　　また，書き出された数字を答えましょう。〔教科書 p135〕

　答え　結果の記号 (　　　)　数字 (　　　)

《ポイント》千の位の数が 0 から 4 までの数字か，5 から 9 までの数字かを考えることで
　　　　　　コンピューターへの四捨五入のしじをつくることができます。
